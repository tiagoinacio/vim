vim
===

Execute the following commands:

$ ln -s vim/.vimrc ~/.vimrc
$ ln -s vim ~/.vim

Optional for Neovim:

$ brew install nvim
$ pip3 install neovim
$ mkdir -p ~/.config/nvim
$ echo 'source ~/.nvimrc' > ~/.config/nvim/init.vim

Optional:

In Ubuntu:
$ sudo apt install fzf

In Mac:
$ brew install fzf
